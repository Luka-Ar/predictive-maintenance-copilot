# RAG System

This directory contains the Retrieval-Augmented Generation (RAG) implementation.

## Components

- **RAG Pipeline** - Retrieves relevant documents from knowledge base before generating explanations
- **Vector Database Integration** - ChromaDB for semantic search
- **Embedding Model** - nomic-embed-text from Ollama

## Purpose

The RAG system enhances LLM explanations by:
1. Converting machine sensor data to embeddings
2. Retrieving similar scenarios from knowledge base
3. Providing context-aware predictions and explanations
